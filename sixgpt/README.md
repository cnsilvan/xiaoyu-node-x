### SIXGPT miner搭建指南


本指南将逐步引导你完成 SIXGPT miner的搭建。

### 前置条件

- Ubuntu 系统原生设备或 Ubuntu 虚拟机设备
- 一个 meta 钱包私钥

### 存储服务器的推荐要求

| 资源          | 规格                             |
|---------------|----------------------------------|
| 推荐操作系统  | Linux 64 位 Intel / AMD          |
| CPU 核心数    | ≥4                               |
| 内存          | ≥ 8 GB                           |
| 带宽          | ≥ 20 Mbps                        |
